# Claude Counter

A lightweight web application that tracks and analyzes Claude AI usage, helping users monitor conversations, token consumption, request patterns, and productivity metrics in real time.

## Features

- 📊 Real-time usage tracking
- 🔢 Token and message counting
- 📈 Visual analytics dashboard
- ⚡ Fast and responsive interface
- 💾 Local data storage
- 🎯 Usage insights and statistics
